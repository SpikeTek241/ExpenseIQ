ExpenseIQ: Smart Expense Tracking Dashboard

ExpenseIQ is a full-stack finanical dashboard that helps users track, analyze, and understand their spending
habits with intelligent insights and a clean modern UI. Built with the main focus on performance, scalability
and real-world fintech use cases.

Features:
-Track and manage transactions
-Smart categorizations of expenses
-Insight into spending patterns
-Fast & Responsive UI
-CRUD Operations (Create, Read, Update, Delete)
-REST API integration

Tech Stack:
Frontend: React(Vite), TypeScript, TailwindCSS

Backend: Node.js, Express.js, Prisma ORM

Database: PostgreSQL
