## 📊 ExpenseIQ — Smart Expense Tracking Dashboard

> A modern full-stack fintech dashboard designed to help users track, analyze, and optimize their spending with real-time insights.

---

### 🚀 Overview
ExpenseIQ is a performance-focused financial dashboard that transforms raw transaction data into meaningful insights. Built with scalability and clean UI in mind, it reflects real-world fintech product design.

---
### ✨ Key Features

- 💳 Track and manage transactions in real-time  
- 🧠 Smart expense categorization  
- 📊 Interactive analytics and spending insights  
- ⚡ Fast, responsive UI (optimized performance)  
- 🔁 Full CRUD functionality  
- 🔌 REST API integration  

---
### 🧰 Tech Stack

**Frontend**
- React (Vite)
- TypeScript
- TailwindCSS  

**Backend**
- Node.js
- Express.js
- Prisma ORM  

**Database**
- PostgreSQL
